Session 3: Getting Comfortable With Free Tools

Hands-on practice with the tools you'll use every day

Self-Paced ~30 minutes

Prerequisites: You should have completed Session 2 before starting this session.

Why This Matters at ATO

Clients don't care which tool you use—they care about quality and speed. But you need to be fluent with at least one tool before Day 1. This session gets you comfortable fast, so you can focus on strategy instead of figuring out the interface.

The goal: hands-on familiarity with the tools that will become your second nature on the job.

What This Session Covers

In this session, you'll get hands-on with real tools. No theory—just doing.

  • Quick setup for Claude, ChatGPT, and Gemini
  • A side-by-side comparison exercise to see how each tool works
  • 10 minutes of free exploration across different use cases
  • A checklist to confirm you're ready to move on

Note: We're focusing on free tools here, pre-placement. Once you're on a client, you may have access to paid versions or other tools.

The Tools You'll Try

Claude (Free)

URL: claude.ai

Who makes it: Anthropic

Why it matters: ATO's recommended tool. Strong at writing, reasoning, and nuance. Great for executive assistant work.

Steps to Get Started

  1. Go to claude.ai
  2. Click "Sign up" (or use your Google/Apple account)
  3. Verify your email
  4. You're in. Start a new conversation

Good to Know

  • Free tier includes the latest Claude 3.5 model
  • You can have multiple conversations; each conversation stays private
  • Clear chat history anytime (button at bottom left)
  • You can upload files and images (great for document review)
  • Attach context: add files, URLs, or photos to your prompt

ChatGPT (Free)

URL: chatgpt.com

Who makes it: OpenAI

Why it matters: Most widely known. General-purpose, fast, intuitive. Good for brainstorming and research.

Steps to Get Started

  1. Go to chatgpt.com
  2. Click "Sign up" (or use your email/Google/Microsoft account)
  3. Verify your email
  4. You're in. Click "Start new chat"

Good to Know

  • Free tier includes GPT-4o model
  • Can browse the web (if enabled in settings)
  • Can analyze files, images, and documents
  • Offers code interpreter for data analysis
  • Sidebar on left lets you organize past conversations

Gemini (Free)

URL: gemini.google.com

Who makes it: Google

Why it matters: Seamless Google integration. Good for research and pulling in real-time info. If your client uses Google Workspace, this is quick.

Steps to Get Started

  1. Go to gemini.google.com
  2. Sign in with your Google account (or create one)
  3. You're in. Start typing in the prompt box

Good to Know

  • Free tier includes the latest Gemini model
  • Can access and analyze Google Docs, Sheets, Gmail (if you authorize)
  • Image generation built-in
  • Search integration for real-time info
  • Integrates with Google Assistant if you use Google Home

The Comparison Exercise (Core of This Session)

The best way to understand these tools is to use them. You're going to run the same prompt in two tools and compare.

Step 1: Pick a Prompt

Choose one of these three options:

Option A: Email Draft (Client Scenario)

"You are an executive assistant. Your client just had a great discovery call with a potential new customer. They want to send a follow-up email to build momentum and suggest a next meeting. Write a 3-4 sentence email that's warm, professional, and doesn't oversell. The tone should feel like a conversation, not a pitch."

Option B: Research Summary

"Summarize the top 5 trends in AI adoption for small businesses in 2025. For each trend, explain what it means and give one practical example. Keep it to 200 words. Use plain language—avoid jargon. Format as a bullet-point list."

Option C: Task Planning

"I'm an executive assistant and my client has a busy week ahead. Help me create a priority list for Monday through Friday. I should focus on: (1) High-impact client-facing work, (2) Strategic projects that move the business forward, (3) Time-sensitive admin tasks. Give me a 5-day breakdown with 3-4 priorities per day. Keep it concise and actionable."

Step 2: Run in Both Tools

Open two browser tabs:

  1. Tab 1: Claude.ai. Paste your prompt. Hit Enter. Wait for the response.
  2. Tab 2: ChatGPT.com (or Gemini.google.com). Paste the same prompt. Hit Enter. Compare.

Step 3: Compare Results

Look for:

  • Structure: How is the response organized? Bullets? Paragraphs? Sections?
  • Tone: Does it feel professional? Casual? Formal? What's the difference between the two tools?
  • Detail: Which tool goes deeper? Which is more concise?
  • Accuracy: If you know the topic, is one tool more accurate or reliable?
  • Length: Which version is longer? Which feels right for your need?

Write a Quick Comparison Note

After comparing, jot down 2-3 sentences on each tool:

  • What did Claude do well?
  • What did ChatGPT/Gemini do well?
  • Which output would you use? Why?

This isn't graded. It's just to help you notice the differences.

Explore Different Use Cases (10 min)

Now that you've run one prompt in both tools, spend 10 minutes playing around. Try different tasks in the same tool and notice how it responds:

  • Draft an email responding to a client's feature request
  • Summarize a topic you're interested in (your notes, a webpage, a concept)
  • Plan your week or a project
  • Brainstorm ideas for a problem you're actually facing
  • Improve your writing: Paste something you wrote. Ask for feedback or to rewrite it
  • Research: Ask about an industry, trend, or skill

Notice what feels natural. Notice what surprises you. This is how you build intuition.

Start Forming a Preference

After exploring, ask yourself:

  • Which interface feels most intuitive to me?
  • Which tool's writing style matches how I want to communicate?
  • Which tool gave me the most usable first-draft output?
  • Do I trust one tool more than the other?
  • If I had to pick one tool to use daily, which would it be?

This is your preference. You don't need to love all three tools equally. You just need one that feels right. And that's usually Claude for ATO team members, but everyone's different.

Practice Exercise (Required)

Time: 15–20 minutes

Your Task

  1. Pick a real task you need to do (or will do soon): drafting something, researching something, planning something
  2. Write a clear prompt for that task using the framework from Session 2
  3. Run it in your preferred tool
  4. Run the same prompt in a different tool
  5. Write a brief comparison note (3-5 sentences):
    • Which tool's output was better for this task?
    • What was the key difference?
    • Would you use this in your real work, or iterate first?
  6. Save your prompt, both outputs, and your comparison note

What We're Looking For

  • A clear, specific prompt (using the Session 2 framework)
  • Evidence that you actually ran the prompt in two tools
  • Honest comparison (not "both were great"—what was actually different?)
  • Outputs that show you engaged with real use cases

Key Takeaways

  1. All three tools work. Pick one, master it, and you're good. (ATO recommends Claude, but you might prefer another.)
  2. Free tools are powerful. You don't need paid versions to do real work. You just need to know how to prompt.
  3. Each tool has a personality. Claude might feel formal; ChatGPT might feel quick; Gemini might feel integrated. Find the one that matches your style.
  4. The prompt matters more than the tool. A great prompt in any tool beats a weak prompt in the "best" tool.
  5. You learn by doing. Exploring and comparing is how you build real fluency. Keep playing.
